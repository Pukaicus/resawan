# Site vitrine – Logiciel Séjours Adaptés

Ce dossier contient un site vitrine statique (HTML/CSS/JS) basé sur l'arborescence :
- Accueil, Le logiciel, Vacanciers, Encadrants, Conformité, Finances, Pour qui ?, Ressources, Démo, Contact
- À propos, Mentions légales & RGPD

## Ouvrir en local
Double-cliquez sur `index.html`.

## Mettre en ligne
Uploadez TOUT le dossier sur votre hébergement (FTP, OVH, etc.).

## Formulaires
Les formulaires "Contact" et "Démo" sont des exemples statiques.
Ils affichent une alerte au lieu d'envoyer un email.
➡️ Vous pouvez les connecter à :
- un script serveur (PHP/Node), 
- un service (Formspree, Make, Zapier),
- votre CRM.

## Personnalisation rapide
- Nom / logo : dans les fichiers HTML (en haut) et dans `assets/css/style.css`
- Couleurs : variables CSS dans `:root`
